# BEMS 建筑能效管理系统 - 专家运维排障知识库

## 故障场景 1：夜间违规耗电 (night_abnormal)
- 触发条件：系统检测到在 22:00 至次日 05:00 期间，单小时总耗电量持续大于 50 kWh。
- 深度原因分析：通常由于公共区域照明未接入时控回路、地下车库排风机未按时启停，或中央空调末端风机盘管阀门卡死导致主机误判负荷而持续低频运转。
- 专家排查建议：
  1. 立即调取大楼安防监控，确认公共走廊及大堂是否有长明灯。
  2. 登录 BAS（楼宇自控系统），强制下发“夜间休眠”指令至所有空调末端。
  3. 安排夜班工程人员巡检 IT 机房，确认是否有非核心服务器在夜间进行高耗能运算。

## 故障场景 2：冷冻水负荷严重异常 (critical_abnormal)
- 触发条件：总电耗激增，且伴随冷水负荷超过历史基线 200%。
- 深度原因分析：极大概率是冷水机组的冷凝器结垢导致换热效率（COP）急剧下降，或者是冷却塔风机皮带断裂导致散热不良。
- 专家排查建议：
  1. 检查冷却水进出水温差，若温差极小，需立即联系维保清洗冷凝器。
  2. 派人上楼顶检查冷却塔运行状态。
