《民用建筑电气设计标准》GB 51348-2019 核心知识库
一、核心术语与定义
一级负荷：中断供电将造成人身伤害、重大损失或重大影响、影响重要用电单位正常工作或造成人员密集公共场所秩序严重混乱的用电负荷。
一级负荷中特别重要负荷：特别重要场所不允许中断供电的用电负荷。
二级负荷：中断供电将造成较大损失或较大影响、影响较重要用电单位正常工作或造成人员密集公共场所秩序混乱的用电负荷。
三级负荷：不属于一级和二级的用电负荷。
应急电源（EPS）：用作应急供电系统组成部分的电源。
不间断电源（UPS）：能够提供满足电子信息设备与计算机系统供电质量要求的不间断供电后备电源装置。
保护接地导体（PE）：用于保护接地的导体。
保护接地中性导体（PEN）：具有保护接地导体和中性导体两种功能的导体。
特低电压（ELV）：相间电压或相对地电压不超过交流均方根值 50V 的电压。
剩余电流保护器（RCD）：能够断开被保护回路所有带电导体，检测并切断剩余电流的保护电器。
等电位联结：为达到等电位，多个可导电部分间的电连接。
雷电电磁脉冲（LEMP）：作为干扰源的雷电流及雷电电磁场产生的电磁场效应。
电涌保护器（SPD）：用于防护雷电电涌侵入电气与电子系统的保护装置。
二、关键设备运行基准与物理阈值
2.1 供配电系统运行红线
一级负荷必须由双重电源供电，一个电源故障时另一个电源不应同时损坏。
一级负荷中特别重要负荷除双重电源供电外，必须增设应急电源供电，应急电源供电回路必须自成系统。
应急电源与正常电源之间必须采取防止并列运行的措施。
150m 及以上超高层公共建筑的消防负荷必须为一级负荷中的特别重要负荷。
住宅小区给水泵房、供暖锅炉房及换热站用电负荷不应低于二级负荷。
正常运行情况下室内照明用电设备端子处电压偏差允许值为 ±5%。
应急照明、景观照明、道路照明和警卫照明用电设备端子处电压偏差允许值为 + 5%、-10%。
一般电动机用电设备端子处电压偏差允许值为 ±5%。
电梯电动机用电设备端子处电压偏差允许值为 ±7%。
变电所计量点的功率因数不宜低于 0.9。
35kV 及以下供配电系统同一电压等级配电级数不宜多于两级，低压系统不宜多于三级。
2.2 变电所设备运行红线
民用建筑内的变压器必须选择干式变压器、气体绝缘变压器或非可燃性液体绝缘变压器。
配电变压器长期工作负载率不宜大于 85%。
变压器低压侧电压为 0.4kV 时，单台变压器容量不宜大于 2000kVA，仅有一台时不宜大于 1250kVA。
预装式变电站干式变压器容量不宜大于 800kVA，油浸式变压器容量不宜大于 630kVA。
成排布置的配电柜长度大于 6m 时，柜后通道必须设置两个出口，两个出口距离大于 15m 时必须增加出口。
长度大于 7m 的配电装置室必须设 2 个出口，长度大于 60m 的配电装置室宜设 3 个出口。
可燃油油浸变压器室耐火等级不得低于二级。
变电所位于地下层或下面有地下层时，通向相邻房间或过道的门必须为甲级防火门。
2.3 自备电源运行红线
应急柴油发电机组市电中断后，低压发电机组必须在 30s 内供电，高压发电机组必须在 60s 内供电。
应急柴油发电机组机房内储油间总储存量不应超过 1m³。
建筑物主体外储油罐容量不应大于 15m³。
EPS 额定输出功率不应小于所连接应急照明负荷总容量的 1.3 倍。
EPS 单机容量不应大于 90kVA。
EPS 用作安全照明电源装置时切换时间不应大于 0.25s。
UPS 额定输出功率应为信息网络设备额定功率总和的 1.2 倍，为其他用电设备供电时应为最大计算负荷的 1.3 倍。
UPS 满载线性负荷时电压畸变率（THDu）应小于或等于 2%。
2.4 低压配电与导体运行红线
消防负荷、导体截面积在 10mm² 及以下的线路必须选用铜芯导体。
固定敷设电力和照明线路铜芯导体最小截面积不应小于 1.5mm²，铝 / 铝合金导体最小截面积不应小于 10mm²。
保护接地导体单独敷设且有防机械损伤保护时，铜导体不应小于 2.5mm²，无防机械损伤保护时铜导体不应小于 4mm²。
手持式及移动式用电设备、室外工作场所用电设备、家用电器回路或插座回路必须设置额定剩余动作电流值不大于 30mA 的剩余电流保护器。
照明系统每一单相分支回路电流不宜超过 16A，所接光源数或 LED 灯具数不宜超过 25 个。
大型建筑组合灯具每一单相回路电流不宜超过 25A，光源数量不宜超过 60 个。
2.5 防雷与接地运行红线
第二类防雷建筑物接闪网网格尺寸不应大于 10m×10m 或 12m×8m。
第三类防雷建筑物接闪网网格尺寸不应大于 20m×20m 或 24m×16m。
第二类防雷建筑物专用引下线间距不应大于 18m，冲击接地电阻不应大于 10Ω。
第三类防雷建筑物专用引下线间距不应大于 25m，冲击接地电阻不应大于 25Ω。
建筑物高度大于 45m 且小于 250m 时，45m 及以上外墙栏杆、门窗等较大金属物必须与防雷装置相连。
建筑物高度大于 60m 时，60m 及以上外墙栏杆、门窗等较大金属物必须与防雷装置相连。
民用建筑各种接地宜采用共用接地装置，接地电阻值应满足最小电阻值要求。
医疗用房内禁止采用 TN-C 接地系统。
浴盆和淋浴场所 0 区内用电设备防护等级必须至少为 IPX7，1 区内防护等级必须至少为 IPX4。
2.6 电气照明运行红线
医院手术室、重症监护室安全照明照度不应低于一般照明照度标准值的 30%。
其他场所安全照明照度不应低于一般照明照度标准值的 10% 且不应低于 15lx。
备用照明照度不应低于该场所一般照明照度标准值的 10%。
长时间视觉工作场所照明光源频闪指数不应大于 10%。
航空障碍标志灯水平安装间距不宜大于 52m，垂直安装自地面 45m 起以不大于 52m 等间距布置。
2.7 常用设备电气运行红线
交流充电桩额定电流不应大于 32A，防水防尘等级不应低于 IP65。
交流充电桩必须选用额定剩余动作电流不大于 30mA 的 A 型剩余电流保护器。
电动机频繁启动时配电母线电压不宜低于额定电压的 90%，不频繁启动时不宜低于额定电压的 85%。
电梯机房地表面照度不应低于 200lx，电梯井道内照度不应低于 50lx。
电辐射供暖每根发热电缆必须单独装设过负荷保护、短路保护及剩余电流动作保护。
三、异常判定规则与排查步骤
3.1 核心状态判定逻辑
用电负荷中断供电造成人身伤害 → 判定为：一级负荷。
用电负荷中断供电造成重大损失或重大影响 → 判定为：一级负荷。
特别重要场所用电负荷不允许中断供电 → 判定为：一级负荷中特别重要负荷。
应急电源与正常电源并列运行 → 判定为：供配电系统严重违规，存在供电安全风险。
民用建筑内使用可燃性液体绝缘变压器 → 判定为：变电所设备配置违规，存在火灾安全风险。
照明分支回路电流超过 16A 或光源数超过 25 个 → 判定为：照明回路过载，存在线路过热风险。
浴盆和淋浴场所 0 区内用电设备防护等级低于 IPX7 → 判定为：潮湿场所电气防护不达标，存在电击风险。
第二类防雷建筑物接闪网网格尺寸大于 10m×10m 或 12m×8m → 判定为：防雷装置不达标，存在雷击风险。
剩余电流保护器额定剩余动作电流大于 30mA 用于手持式用电设备 → 判定为：电击防护失效，存在人身触电风险。
3.2 故障排查步骤
故障：一级负荷供电中断
检查双重电源是否同时发生故障，确认单电源故障状态。
检查应急电源是否自动投入，确认应急电源切换装置动作状态。
检查应急电源供电回路是否接入其他负荷，确认回路独立性。
检查电源监测和故障报警装置是否正常工作，确认故障信号反馈状态。
故障：低压配电线路短路
检查线路保护电器分断能力是否小于安装处预期短路电流，确认保护选型匹配性。
检查导体截面积是否满足短路热稳定要求，确认导体绝缘温度限值。
检查上下级保护电器是否具备选择性，确认故障切断精准性。
故障：防雷装置失效
检查接闪器、引下线、接地网连接是否牢固，确认电气通路完整性。
检查引下线间距是否超过规范限值，确认防雷保护覆盖范围。
检查接地电阻值是否超过规范要求，确认接地装置有效性。
检查金属构件与防雷装置等电位联结是否可靠，确认反击风险防控状态。
故障：潮湿场所电击风险
检查场所内剩余电流保护器额定动作电流是否大于 30mA，确认保护参数合规性。
检查用电设备防护等级是否满足区域要求，确认设备防护有效性。
检查辅助等电位联结是否完整实施，确认可导电部分等电位状态。
检查供电回路是否采用 SELV 或 PELV 保护措施，确认低电压防护状态。
故障：应急照明无法正常切换
检查 EPS 切换时间是否超过规范限值，确认切换装置响应速度。
检查 EPS 额定输出功率是否满足负荷要求，确认设备容量匹配性。
检查应急照明供电回路是否与正常照明回路独立，确认回路独立性。
检查应急照明灯具是否为瞬时点燃光源，确认光源启动特性。
