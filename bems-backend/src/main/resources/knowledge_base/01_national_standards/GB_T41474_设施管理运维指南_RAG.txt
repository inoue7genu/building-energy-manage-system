GB/T 41474-2022《设施管理 运作与维护指南》核心知识库
一、核心术语与定义
设施管理：在建筑环境内整合人员、场所、过程，以改善人们生活质量、提高核心业务生产力为目的的组织职能。
服务水平协议：需求组织和服务供应商之间就服务交付的绩效、测量和条件所约定的文件。
关键绩效指标：为绩效考评提供必不可少信息的度量参数。
纠正性维护：在故障识别之后进行，旨在使设施能够达到所需功能状态的一系列活动。
预测性维护：计划内的、利用科技通过监测和分析设备情况来预报设备故障的一系列活动。
预防性维护：基于预定周期或者根据预定指标，以降低故障发生几率或者功能下降而开展的一系列活动。
设施状态指标：设施积压缺陷维修所需的成本除以资产重置成本得出的评估指标。
计算机化维护管理系统：以计算机辅助用于帮助计划、管理和跟踪维护活动的平台工具。
以可靠性为中心的维护：以设备可靠性及设备故障后果为主要依据，整合多种维护方式的优化维护策略。
二、关键设备运行基准与物理阈值
2.1 设施状态指标阈值
设施状态指标小于或等于 5% → 设施状态定义为优秀。
设施状态指标 5%~10% → 设施状态定义为可接受。
设施状态指标 10%~65% → 设施状态定义为较差。
设施状态指标大于或等于 65% → 设施状态定义为严重。
2.2 设施维护策略配比阈值
A 类设施维护配比 → 90% 计划性维护，10% 非计划性维护。
C 类设施维护配比 → 90% 非计划性维护。
D 类设施维护方式 → 仅采用事后维修。
2.3 服务绩效阈值
预防性维护工单按时完成率大于 98% → 维护绩效判定为合格。
预防性维护工单按时完成率 95%~98% → 维护绩效判定为部分合格。
预防性维护工单按时完成率小于 95% → 维护绩效判定为不合格。
维修工单在服务水平协议约定时间内解决完成率大于 98% → 维修绩效判定为合格。
三、异常判定规则与排查步骤
3.1 核心状态判定逻辑
设施故障或停机导致重大安全环境影响与核心业务重大中断 → 判定为 A 类关键设施。
设施故障或停机对生产、安全产生较大影响 → 判定为 B 类关键设施。
设施故障或停机对生产、安全无较大影响 → 判定为 C 类设施。
设施故障或停机对生产、安全无影响 → 判定为 D 类设施。
非计划性维护占比过高 → 判定为设施维护策略不合理，成本与风险上升。
3.2 故障排查步骤
故障：设施非计划性维护占比超标
核查 A 类、B 类关键设施的预防性维护计划执行率。
检测关键设施运行参数，识别预测性维护监测缺失项。
评估设施状态指标，确定缺陷维修滞后项。
调整设施分级匹配度，优化维护策略配比。
故障：设施服务未达成服务水平协议要求
核对服务水平协议约定的响应时间、完成率等量化指标。
检查服务供应商日常执行记录与关键控制点合规性。
核查设施运行参数与服务交付标准的匹配情况。
落实不符合项纠正措施，更新服务考核指标。
