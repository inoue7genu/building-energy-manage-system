《建筑电气与智能化通用规范》GB 55024-2022 核心知识库
一、核心术语与定义
建筑能效监管系统：对建筑用能系统与设备的能耗数据进行监测、统计、分析、监管的智能化系统。
建筑设备管理系统：支持开放式技术，具备自诊断、故障隔离、故障报警、参数超限报警与保护动作反馈功能的建筑设备监控与管理系统。
应急电源：正常电源断电时，在约定时间内维持重要安全设施电气装置或电气设备运行所需的电源。
保护接地导体 (PE)：用于电气设备外露可导电部分保护接地的安全保护导体。
剩余电流动作保护电器：规定条件下剩余电流达到整定值时能自动分断电路，用于电击附加防护的机械开关电器或组合电器。
安全特低电压 (SELV)：正常条件下不接地，电压不超过交流 50V 或直流 120V 的电气系统电压。
等电位联结：为实现电位均衡，将建筑物内可导电部分、接地导体、总接地端子进行电气连接的措施。
电涌保护器：安装于进出防雷建筑物的线路上，用于防护雷电波侵入的过电压保护装置。
二、关键设备运行基准与物理阈值
2.1 建筑能效监管系统运行红线
建筑能效监管系统的设置不得影响用能系统与设备的功能，不得降低用能系统与设备的技术指标。
建筑能效监管系统应建立信息数据库，具备根据需求形成运行记录的功能。
2.2 建筑设备管理系统运行红线
建筑设备管理系统应支持开放式系统技术，具备系统自诊断、故障部件自动隔离、自动唤醒、故障报警及自动监控功能。
建筑设备管理系统应具备参数超限报警和执行保护动作的功能，并反馈动作信号。
地下机动车库配套的建筑设备管理系统应设置与排风设备联动的一氧化碳浓度监测装置。
通风空调系统电加热器配套的建筑设备管理系统应具备电加热器与送风机连锁、电加热器无风断电、超温断电保护及报警监控功能，具备风机系统延时运行后停机监控功能。
2.3 低压配电与电击防护设备阈值
安全特低电压配电系统的电压不得超过交流 50V 或直流 120V。
用于电击附加防护的剩余电流动作保护电器额定剩余电流动作值不得大于 30mA。
TN 系统 220V、额定电流不超过 63A 的电源插座回路切断电源的最长时间为 0.4s。
TT 系统 220V、额定电流不超过 63A 的电源插座回路切断电源的最长时间为 0.2s；采取保护等电位联结时切断电源的最长时间为 0.4s。
装有浴盆 / 淋浴场所 0 区内电气设备应采用不超过交流 12V 或直流 30V 的安全特低电压供电。
游泳池 0 区和 1 区内电气设备应采用不超过交流 12V 或直流 30V 的安全特低电压供电。
2.4 照明设备阈值
室外灯具防护等级不应低于 IP54，埋地灯具防护等级不应低于 IP67，水下灯具防护等级不应低于 IP68。
正常照明灯具安装高度 2.5m 及以下且采用交流低压供电时，应设置剩余电流动作保护电器作为附加防护。
疏散照明和疏散指示标志灯安装高度 2.5m 及以下时，应采用安全特低电压供电。
质量大于 10kg 的灯具，固定装置和悬吊装置应按灯具质量 5 倍恒定均布荷载做强度试验，持续时间不得少于 15min。
2.5 防雷与接地设备阈值
第二类防雷建筑物专用引下线的平均间距不应大于 18m。
第三类防雷建筑物专用引下线的平均间距不应大于 25m。
接闪带采用热镀锌圆钢或扁钢制成时，截面面积不应小于 50mm²。
接闪杆采用热镀锌圆钢制成时，直径不应小于 20mm；采用热镀锌钢管制成时，直径不应小于 40mm。
智能化系统单独设置的接地线应采用截面面积不小于 25mm² 的铜材。
单独敷设的铜质保护接地导体有机械损伤防护时，截面面积不应小于 2.5mm²；无机械损伤防护时，截面面积不应小于 4mm²。
2.6 布线系统阈值
导管和电缆槽盒内配电电线的总截面面积不应超过导管或电缆槽盒内截面面积的 40%。
电缆槽盒内控制线缆的总截面面积不应超过电缆槽盒内截面面积的 50%。
室内干燥场所金属导管布线的壁厚不应小于 1.5mm。
室内潮湿场所金属导管明敷的壁厚不应小于 2.0mm。
金属电缆桥架全长不大于 30m 时，不应少于 2 处与保护导体可靠连接；全长大于 30m 时，每隔 20m~30m 应增加一个连接点。
三、异常判定规则与排查步骤
3.1 核心状态判定逻辑
建筑能效监管系统影响用能系统功能或降低设备技术指标 → 判定为建筑能效监管系统设置违规。
建筑设备管理系统不具备自诊断、故障报警、参数超限保护反馈功能 → 判定为建筑设备管理系统功能缺失。
地下机动车库建筑设备管理系统未设置一氧化碳浓度监测联动装置 → 判定为建筑设备管理系统配置缺失。
剩余电流动作保护电器额定剩余电流动作值大于 30mA → 判定为剩余电流动作保护电器电击防护能力不达标。
灯具安装高度 2.5m 及以下未按要求设置剩余电流保护或采用安全特低电压 → 判定为照明系统电击防护措施缺失。
第二类防雷建筑物专用引下线平均间距大于 18m → 判定为防雷引下线设置违规。
智能化系统接地线铜材截面面积小于 25mm² → 判定为智能化系统接地导体截面面积不达标。
导管内配电电线总截面面积超过导管内截面面积 40% → 判定为布线系统散热条件不达标。
3.2 故障排查步骤
故障：建筑能效监管系统数据异常
检查建筑能效监管系统与用能设备的通信接口连接状态。
核查建筑能效监管系统能耗采集装置安装是否干扰用能计量设备。
校验建筑能效监管系统数据库运行记录生成功能。
故障：建筑设备管理系统联动失效
检查建筑设备管理系统与被控设备的控制信号传输线路。
测试建筑设备管理系统自诊断、故障隔离与报警模块运行状态。
核查电加热器与送风机连锁、无风断电、超温断电保护逻辑设置。
检查地下车库一氧化碳浓度监测装置与排风设备联动触发参数。
故障：剩余电流动作保护电器拒动
测试剩余电流动作保护电器额定剩余电流动作值是否大于 30mA。
检查保护接地导体 (PE) 连接是否可靠，无串联断开情况。
验证剩余电流动作保护电器试验按钮操作与动作时间是否符合要求。
故障：防雷装置接地电阻超标
检查防雷引下线与接闪器、接地装置的连接是否牢固无锈蚀。
核查防雷接地装置与建筑物基础钢筋的连接状态。
检测等电位联结导体连接是否可靠，截面面积符合规范要求。
故障：布线系统绝缘故障
检测线缆绝缘电阻值，低压配电线路绝缘电阻不应小于 1MΩ。
检查金属导管、电缆桥架与保护导体的连接点数量与牢固度。
核查线缆穿越变形缝处是否设置补偿装置。
