SLS 系列单级单吸立式离心泵核心知识库
一、核心术语与定义
SLS 系列单级单吸立式离心泵：遵循 ISO2858 国际标准与最新国家标准，以 IS 型离心泵性能数据为基础设计的节能立式离心泵，可替代 IS 卧式泵、DL 型泵等普通泵。
SLSD 型泵：SLS 系列衍生型单级单吸立式离心泵，适配小流量低扬程输送场景。
SLR 型泵：同 SLS 系列结构的单级单吸立式热水泵，适配高温热水输送场景。
SLH 型泵：同 SLS 系列结构的单级单吸立式化工泵，适配腐蚀性液体输送场景。
SLY (YG) 型泵：同 SLS 系列结构的单级单吸立式防爆油泵，适配易燃易爆无腐蚀液体输送场景。
SLHY 型泵：同 SLS 系列结构的单级单吸立式防爆化工泵，适配易燃易爆腐蚀性液体输送场景。
二、关键设备运行基准与物理阈值
2.1 系统工作压力阈值
SLS 系列单级单吸立式离心泵系统最大工作压力为 1.6MPa，吸入压力与扬程之和≤1.6MPa。
SLS 系列单级单吸立式离心泵静压试验压力为 2.5MPa。
SLS 系列单级单吸立式离心泵系统工作压力＞1.6MPa 时，过流部分与联接部分需采用铸钢材料。
2.2 输送介质阈值
SLS 系列单级单吸立式离心泵输送介质为清水或物理性质类似清水的无腐蚀液体。
SLS 系列单级单吸立式离心泵输送介质中固体不溶物体积≤单位体积 0.1%，粒度＜0.2mm。
SLS 系列单级单吸立式离心泵输送介质温度≤80℃。
SLR 型单级单吸立式热水泵输送介质温度≤120℃。
SLH 型、SLY (YG) 型、SLHY 型单级单吸立式泵输送介质温度范围为 - 20℃~120℃。
SLY (YG) 型单级单吸立式防爆油泵输送介质密度 ρ＜1000kg/m³。
2.3 环境条件阈值
SLS 系列单级单吸立式离心泵环境温度≤40℃。
SLS 系列单级单吸立式离心泵安装海拔高度≤1000m。
SLS 系列单级单吸立式离心泵相对湿度≤95%。
2.4 运行转速阈值
SLS 系列单级单吸立式离心泵常规运行转速为 2950r/min、1480r/min、980r/min。
三、异常判定规则与排查步骤
3.1 核心状态判定逻辑
SLS 系列单级单吸立式离心泵系统工作压力＞1.6MPa → 判定为超压运行，过流与联接部件存在破损风险。
SLS 系列单级单吸立式离心泵输送介质固体粒度≥0.2mm → 判定为介质不达标，叶轮与泵体存在磨损风险。
SLS 系列单级单吸立式离心泵输送介质温度＞80℃ → 判定为超温运行，机械密封与泵轴存在损坏风险。
SLS 系列单级单吸立式离心泵环境温度＞40℃ → 判定为环境超标，电机与轴承存在过热烧毁风险。
3.2 故障排查步骤
故障：SLS 系列单级单吸立式离心泵密封渗漏
检查 SLS 系列单级单吸立式离心泵轴用硬质合金机械密封是否磨损、破损。
检查 SLS 系列单级单吸立式离心泵输送介质粒度是否≥0.2mm、温度是否超标。
拆卸 SLS 系列单级单吸立式离心泵泵盖螺母，取出电机及传动组件更换机械密封。
故障：SLS 系列单级单吸立式离心泵运行振动超标
检查 SLS 系列单级单吸立式离心泵叶轮动静平衡是否达标。
检查 SLS 系列单级单吸立式离心泵重心是否与泵脚中心重合。
检查 SLS 系列单级单吸立式离心泵安装底脚是否固定牢固。
故障：SLS 系列单级单吸立式离心泵流量扬程不足
核对 SLS 系列单级单吸立式离心泵型号对应的额定流量、扬程参数是否匹配工况。
检查 SLS 系列单级单吸立式离心泵叶轮是否存在堵塞、磨损情况。
检查 SLS 系列单级单吸立式离心泵运行转速是否达到额定转速。
